# Radio Sathi

Blind-first Android radio companion for voice-controlled listening.

## Build
Open this folder in Android Studio. Let Gradle sync, then use:

Build > Build Bundle(s) / APK(s) > Build APK(s)

## Commands
- play one
- play two
- list favourites
- next
- previous
- pause
- resume
- stop
- current channel
- play Kishore Kumar on YouTube

## Important
This project opens YouTube through the official YouTube app or browser. It does not extract hidden YouTube audio.
The stream URLs in `app/src/main/assets/channels.json` are candidate AIR/Prasar Bharati stream URLs and may need verification/replacement if any stream changes.
